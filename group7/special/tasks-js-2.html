1.
В Україні максимально дозволена швидкість у населених пунктах 50 км/год. Відомо, з якою швидкістю рухається автомобіль Івана.
Чи порушує він правила?
Дано:
швидкість автомобіля Івана
Результат:
чи є порушення правил дорожнього руху
45==>Не порушує правила
92==>Порушує правила
50==>Не порушує правила
Розв’язок (спосіб 1)
<script>

  let speed = Number(prompt("Яка швидкість авто?"));
  let violation;

  if (speed > 50)
    violation = "Порушує правила";
  else
    violation = "Не порушує правила";

  alert(violation);

</script>
Розв’язок (спосіб 2)
<script>

  let speed = Number(prompt("Яка швидкість авто?"));
  let violation;

  if (speed > 50)
    violation = "Порушує правила";
  if (speed <= 50)
    violation = "Не порушує правила";

  alert(violation);

</script>
